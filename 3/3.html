Componentes del Sistema:
Un pequeño sensor que dura hasta 14 días y se aplica en la parte posterior del brazo. Es resistente al agua y por lo tanto se puede usar cuando uno se baña, nada o hace ejercicio (hasta 1 metro de profundidad y hasta 30 minutos).
El lector FreeStyle Libre se utiliza para escanear el sensor. Con cada escaneo, el usuario verá:

Nivel de glucosa actual.

Un historial de 8 horas

Una flecha de tendencia que muestra si la glucosa está subiendo, bajando o permanece estable.

El sensor debe escanearse por lo menos 1 vez cada 8 horas para no perder información de las mediciones de glucosa.
